Huke

The best and worst programming language ever made.

Download .zip Download .tar.gz View on GitHub

Huke (Hard to Understand Puke) is an esoteric programming language written in pure, standard C, and based of the Befunge. It's similiar to Malborge, Parenthetic, and all those other little (and weird) little language we know and love.

Installing

Pretty simple:

$ git clone https://github.com/beakr/huke.git
$ make

Writing some Huke scripts

Huke is a very messy, weird language, but very fun to mess around with. Many programmers use languages like Befunge just to see what they can make out of complex symbols and patterns. Here's a simple program that prints "Hello" in Huke:

"!olleH",,,,,,@

The string procedure "!olleH", which has six characters in it (declared by the commas), is printed and then @ is used to terminate the program. Once you run make you can just run ./huke program.hk to run your simple Huke script.

For some more scripts I highly recommend checking out the /samples directory in the Github repo. For a Befunge reference, check out the Wikipedia article.
